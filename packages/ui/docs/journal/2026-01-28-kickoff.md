**Holocron Portal** is a data documentation platform with:
- Next.js 15 (App Router)
- TypeScript (strict mode)
- shadcn/ui + Tailwind CSS
- TanStack Query v5
- holocron-ts SDK
- Bun runtime

### Development Principles

From `CLAUDE.md`:
- 🧪 **TDD** - Tests first, always
- 🔒 **Strict TypeScript** - No `any`, typed errors
- 🐳 **Containerized** - Docker only, nothing on host
- ⚡ **Server-first** - Server components by default
- 📦 **SDK-only** - Never call Holocron API directly
