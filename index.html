A graph of every dataset, owner, and rule
in your organisation.

Holocron is a declarative data governance platform — REST API on Neo4j, a TypeScript SDK, a Next.js portal with a 3D galaxy map, an MCP server for AI assistants, and a plugin ecosystem to keep the catalog in sync with reality.

Get startedRead the conceptsView on GitHub

One graph for everything

Datasets, reports, processes, systems, people, teams, rules — all in one Neo4j graph with multi-label nodes and clear relationship types.

Hybrid search

Vector + fulltext fused per kind, with a small DSL (ds:, owner:, feeds:, "phrase", -exclude) for power users.

3D galaxy map

Every asset is a star, every relation an edge. Animated lineage particles, multi-lock focus, keyboard nav.

Plugin ecosystem

Connectors for CSV, Excel, Postgres, Power BI. Exporters for Excel and Markdown. Audit, compliance, and PII detection. All discovered via entry points.

Outbound webhooks

Subscribe to event topics. HMAC-SHA256 signed deliveries, fire-and-forget, auto-disable on failures.

MCP-ready

An MCP server exposes the catalog to Claude Desktop, Claude Code, and any MCP client. Browse, edit, run plugins by chat.

Run it locally

Three services, one command. Podman or Docker — no host installs.

make up        # neo4j + api + ui
make health    # smoke check
# API  → http://localhost:8100
# UI   → http://localhost:3333
# Neo4j → http://localhost:7474
