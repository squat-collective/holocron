14:T543b,<h1 id="ui-extension-framework">UI extension framework<a class="anchor" aria-label="Link to section" href="#ui-extension-framework">#</a></h1>
<blockquote>
<p>Every command in the ⌘K palette comes from an extension. Pages contribute commands; built-ins handle creation, navigation, sharing, governance, dev tools, and plugin invocation.</p>
</blockquote>
<p>This is a UI-only concept — distinct from the API plugins (<a href="/docs/plugins">plugins.md</a>). Extensions don't run on the server; they're just functions that build commands from the current page context.</p>

<p><span data-rehype-pretty-code-figure=""><code data-language="plaintext" data-theme="github-dark-dimmed github-light"><span data-line=""><span>plugins-adapter.tsx</span></span></code></span> bridges the API plugin registry into the palette: every plugin shows up as a command (<span data-rehype-pretty-code-figure=""><code data-language="plaintext" data-theme="github-dark-dimmed github-light"><span data-line=""><span>Run: &#x3C;plugin name></span></span></code></span>), grouped by capability (Import / Export). Selecting a command opens a wizard that auto-renders inputs from the plugin's manifest, then <span data-rehype-pretty-code-figure=""><code data-language="plaintext" data-theme="github-dark-dimmed github-light"><span data-line=""><span>POST</span></span></code></span>s to <span data-rehype-pretty-code-figure=""><code data-language="plaintext" data-theme="github-dark-dimmed github-light"><span data-line=""><span>/api/holocron/plugins/{slug}/run</span></span></code></span>.</p>
<h2 id="conventions">Conventions<a class="anchor" aria-label="Link to section" href="#conventions">#</a></h2>
<ul>
<li>Extensions are <strong>pure functions</strong> — no React, no hooks, no I/O at compute time. Side effects happen inside <span data-rehype-pretty-code-figure=""><code data-language="plaintext" data-theme="github-dark-dimmed github-light"><span data-line=""><span>run()</span></span></code></span>.</li>
<li><span data-rehype-pretty-code-figure=""><code data-language="plaintext" data-theme="github-dark-dimmed github-light"><span data-line=""><span>ExtensionContext.queryClient</span></span></code></span> can be <span data-rehype-pretty-code-figure=""><code data-language="plaintext" data-theme="github-dark-dimmed github-light"><span data-line=""><span>null</span></span></code></span> during initial hydration — always check before calling <span data-rehype-pretty-code-figure=""><code data-language="plaintext" data-theme="github-dark-dimmed github-light"><span data-line=""><span>invalidate()</span></span></code></span>.</li>
<li>Use <span data-rehype-pretty-code-figure=""><code data-language="plaintext" data-theme="github-dark-dimmed github-light"><span data-line=""><span>group:</span></span></code></span> to keep the palette scannable. Existing groups: <span data-rehype-pretty-code-figure=""><code data-language="plaintext" data-theme="github-dark-dimmed github-light"><span data-line=""><span>Navigation</span></span></code></span>, <span data-rehype-pretty-code-figure=""><code data-language="plaintext" data-theme="github-dark-dimmed github-light"><span data-line=""><span>Create</span></span></code></span>, <span data-rehype-pretty-code-figure=""><code data-language="plaintext" data-theme="github-dark-dimmed github-light"><span data-line=""><span>Asset</span></span></code></span>, <span data-rehype-pretty-code-figure=""><code data-language="plaintext" data-theme="github-dark-dimmed github-light"><span data-line=""><span>Actor</span></span></code></span>, <span data-rehype-pretty-code-figure=""><code data-language="plaintext" data-theme="github-dark-dimmed github-light"><span data-line=""><span>Rule</span></span></code></span>, <span data-rehype-pretty-code-figure=""><code data-language="plaintext" data-theme="github-dark-dimmed github-light"><span data-line=""><span>Governance</span></span></code></span>, <span data-rehype-pretty-code-figure=""><code data-language="plaintext" data-theme="github-dark-dimmed github-light"><span data-line=""><span>Plugins</span></span></code></span>, <span data-rehype-pretty-code-figure=""><code data-language="plaintext" data-theme="github-dark-dimmed github-light"><span data-line=""><span>Pins</span></span></code></span>, <span data-rehype-pretty-code-figure=""><code data-language="plaintext" data-theme="github-dark-dimmed github-light"><span data-line=""><span>Recents</span></span></code></span>, <span data-rehype-pretty-code-figure=""><code data-language="plaintext" data-theme="github-dark-dimmed github-light"><span data-line=""><span>Dev</span></span></code></span>.</li>
<li>Commands are deduplicated by <span data-rehype-pretty-code-figure=""><code data-language="plaintext" data-theme="github-dark-dimmed github-light"><span data-line=""><span>id</span></span></code></span>; namespace them like <span data-rehype-pretty-code-figure=""><code data-language="plaintext" data-theme="github-dark-dimmed github-light"><span data-line=""><span>my-thing.copy-name</span></span></code></span>.</li>
<li>Don't fetch in <span data-rehype-pretty-code-figure=""><code data-language="plaintext" data-theme="github-dark-dimmed github-light"><span data-line=""><span>computeCommands</span></span></code></span> — read from React Query cache via <span data-rehype-pretty-code-figure=""><code data-language="plaintext" data-theme="github-dark-dimmed github-light"><span data-line=""><span>queryClient.getQueryData()</span></span></code></span> instead.</li>
</ul>
<h2 id="why-extensions-not-just-inline-buttons">Why extensions, not just inline buttons?<a class="anchor" aria-label="Link to section" href="#why-extensions-not-just-inline-buttons">#</a></h2>
<ul>
<li>One palette to learn (⌘K), one search, one keyboard model.</li>
<li>Commands are dynamic: <span data-rehype-pretty-code-figure=""><code data-language="plaintext" data-theme="github-dark-dimmed github-light"><span data-line=""><span>pii-detector</span></span></code></span> can suggest "Apply to email field" only when an <span data-rehype-pretty-code-figure=""><code data-language="plaintext" data-theme="github-dark-dimmed github-light"><span data-line=""><span>email</span></span></code></span>-named field is on screen.</li>
<li>Plugin authors get a UI for free — the plugin adapter exposes their plugin without the UI team writing a button.</li>
<li>Users discover features they didn't know existed by typing what they want.</li>
